PhypInfo

Show source in phyp.py:27

Represents a monitor sample from the PHYP monitor.

The PHYP PCM monitor JSON data can be parsed by this. The base structure is: - PhypInfo - Info - PhypSample - PhypSystemFirmware - PhypSystemProcessor - PhypSystemMemory - PhypSharedProcPool - PhypVMSample (list - client virtual machines) - PhypLparProc - PhypLparMemory - PhypNetwork - PhypVEA - PhypSriovLparPort - PhypStorage - PhypStorageVAdpt - PhypVirtualFCAdpt - PhypViosSample (list - Virtual I/O Servers) - PhypLparProc
